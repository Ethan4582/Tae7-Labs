# Tae7-Labs

A curated library of premium, high-performance UI components built with React, Next.js, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- **Performance First**: Optimized animations and transitions.
- **Modern Stack**: Built with Next.js 15+, React 19, and Tailwind CSS 4.
- **Ready to Use**: Copy-paste components designed for modern web apps.
